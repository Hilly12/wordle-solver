## Wordle Solver

For direct usage
```
python wordle.py -w "crane" "quake" -c "**g*g" "**ggg"
```

For interactive usage
```
python wordle.py
```

For additional options
```
python wordle.py -h
```

Run from anywhere
```
git clone https://github.com/Hilly12/wordle-solver.git && cd wordle-solver && python wordle.py && cd .. && rm -rf wordle-solver
```

Dependencies
```
numpy
```
